# Wallet Logos

This directory should contain the official wallet logo images for the wallet selection modal.

## Required Files

Please download the official logos for each wallet and save them with these exact filenames:

1. **suiet.png** - Suiet Wallet logo
   - Source: https://suiet.app/ or Chrome Web Store
   - Recommended size: 128x128px or higher (PNG with transparency)

2. **phantom.png** - Phantom wallet logo
   - Source: https://phantom.app/ or their brand assets
   - Recommended size: 128x128px or higher (PNG with transparency)

3. **martian.png** - Martian Sui Wallet logo
   - Source: https://martianwallet.xyz/ or Chrome Web Store
   - Recommended size: 128x128px or higher (PNG with transparency)

4. **okx.png** - OKX Wallet logo
   - Source: https://www.okx.com/web3 or their brand assets
   - Recommended size: 128x128px or higher (PNG with transparency)

5. **glass.png** - Glass Wallet logo
   - Source: https://glass.xyz/ or their brand assets
   - Recommended size: 128x128px or higher (PNG with transparency)

6. **slush.png** - Slush Wallet logo
   - Source: https://slushwallet.com/ or their brand assets
   - Recommended size: 128x128px or higher (PNG with transparency)

## How to Download

1. Visit each wallet's official website
2. Look for their logo/brand assets page
3. Download the logo in PNG format with transparency
4. Resize to 128x128px or keep original if it's square
5. Save with the exact filename listed above

## Alternative Sources

- Chrome Web Store: Right-click on wallet extension icon → "Inspect" → Find image URL
- GitHub: Some wallets have their logos in their GitHub repositories
- Simple Icons: https://simpleicons.org/ (may have some wallet logos)

## Notes

- The component will automatically fall back to emoji icons if the image files are not found
- Images should have transparent backgrounds for best appearance
- PNG format is recommended for best quality
